# Onboarding — Start Here

Welcome aboard! This section takes you from a fresh laptop to your first merged
pull request. Work through it in order:

1. **[Day-1 Checklist](/onboarding/day-one)** — accounts, access, and tools you
   need before anything else.
2. **[Dev Environment Setup](/onboarding/environment)** — get the codebases
   running locally.
3. **[Your First Week](/onboarding/first-week)** — what to read, who to meet,
   what's expected.
4. **[Your First PR](/onboarding/first-pr)** — ship something small, end to end.

**Goal: your first PR merged within your first week.** If anything in this
section is wrong or unclear, fixing the docs is a great first PR.
