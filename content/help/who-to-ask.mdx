# Who to Ask

Organized by problem, not by org chart.

| Situation | Go to |
| --------- | ----- |
| I need access to GitHub / Trello / Figma / Vercel / password manager | CTO (Seif) |
| My PR needs review / I'm blocked on a technical decision | CTO (Seif) |
| Is this design final? Where are the tokens for X? | UI/UX designer |
| I want to request new work for the department | The [requests channel](/how-we-work/communication) — never DMs |
| This is genuinely urgent and must jump the queue | **Operations manager** (who weighs it against what it delays) |
| Production is broken / I may have leaked a credential | CTO (Seif), **immediately** — see [Security Basics](/standards/security) |
| Something in this handbook is wrong | Open a PR (footer link on every page) — you don't need permission to fix docs |

Current people behind these roles are listed in the
[Team Map](/about/team).
