# Security Basics

The non-negotiables. These exist to protect our users and the business.

- **Never commit secrets.** Env vars live in the hosting platform / a password
  manager — never in git, never in chat.
- **Never paste production data** into third-party tools (including AI tools)
  unless the tool is approved for it.
- **All input is untrusted.** Validate on the server, not just the client.
- **Dependencies:** keep them updated; investigate audit warnings before release.
